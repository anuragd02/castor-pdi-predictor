# CASTOR PDI Predictor 🌿

An AI-powered forecasting and advisory system for predicting **Botrytis Gray Mould (BGM)** severity in castor crops using weather-based machine learning models.

The system analyzes environmental parameters such as:

* Temperature
* Relative Humidity
* Rainfall
* Rainy Days
* Wind Speed
* Sunshine Hours

to predict the **Percent Disease Index (PDI)** and classify disease risk into:

* Low
* Moderate
* High
* Very High

Based on the predicted severity, the platform provides:

* Disease risk assessment
* Fungicide recommendations
* Preventive advisory actions
* Field monitoring guidance

Built using:

* Python
* Streamlit
* Scikit-learn
* NumPy
* Joblib

The project focuses on enabling early disease forecasting, reducing crop loss, and supporting data-driven agricultural decision-making through an interactive advisory dashboard.
